Documentação do Aplicativo de Gerenciamento de Desempregados

Onde vamos criar um aplicativo de controle, que permitira saber a quantidade de desempregados. O trabalho foi discutido entre os membros, o trabalho fico da seguinte forma um deve fazer a parte do Documentário para por no HTML e outro O UML 10/05/2024.

1. Objetivo do Projeto

Desenvolver um aplicativo de gerenciamento de desempregados para fornecer dados estatísticos e facilitar o controle das informações relacionadas ao desemprego.

2. Tecnologia Utilizada

Utilizaremos Python como linguagem de programação principal devido à sua versatilidade e ampla gama de bibliotecas. Faremos uso de operações CRUD (Create, Read, Update, Delete) para gerenciar os dados dos desempregados de forma eficiente.

3. Modelagem de Dados

Cada desempregado pode ter atributos como nome, idade, cidade, área de especialização, experiência, habilidades, salário desejado, e status de empregabilidade.

4. Persistência de Dados

Você pode armazenar os dados dos desempregados em um banco de dados SQLite integrado ao Python. Outra opção é em um banco de dados NoSQL como MongoDB.

5. Testes

Escreva testes para garantir que todas as operações CRUD funcionem conforme o esperado e que os dados sejam armazenados e recuperados corretamente.
